Detailed Description

Logger for a standard stream. Log to a given FILE pointer, writing log messages immediately. Color and lack of seek are assumed. The stream is flushed but the file is not closed at destruction time. No check that the file is still open at any point is performed. It is recommended to use this only for stdout or stderr.
